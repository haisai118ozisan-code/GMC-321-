低圧時注意

切り取り波形

電圧を下げるほど、波形の「お尻」の方しか使わなくなります。出力100.0V（200Vの約50%）の状態では、スイッチのタイミングが非常にシビアになり、オン・オフが乱れやすくなります。

なぜ100V以下（50%以下）で不安定に見えるのか？

● 制御の細かさ（分解能）
200V機は0〜200Vを制御するように作られています。100V以下で使うということは、常に「能力の半分以下」で微調整を強いている状態です。高級車のアクセルをごく僅かに踏み続けて時速5kmを維持しようとするような、ギクシャクした動きになりやすいのです。
● サイリスタの特性
サイリスタは「一度オンになると、電流がゼロになるまでオフにならない」性質があります。電圧を絞る＝オンにする時間を遅らせる ということですが、あまりに遅くすると、オンになった瞬間にサイクルが終わってしまい、安定して電力を伝えられなくなります。
▼ 訂正・更新（伸興技術部 試験データ T09293 を反映）

前回「トルクモーター × インバータ = NG」と説明しましたが、これは不正確でした。
伸興技術部の試験データ（B2OT-450 周波数とトルク相関）によれば、低速で大トルクが必要な用途では、インバータ駆動の方が圧倒的に有利です。
本ページは試験データに基づき、正しい結論に書き換えています。

結論（試験データ T09293 ベース）

低速駆動なら「インバータ + V/f制御」の方が トルクが約4倍 / 電流1/4

伸興 B2OT-450 トルクモーター（5N-m / 4P / 60Hz / 220V）の実測データでは：
サイリスタ式で 60Hz / 110V に絞っても十分なトルクが出ない 領域でも、
インバータで 30Hz / 110V（V/f比一定）に下げれば、約2.7N-m が安定して取れる。
さらに電流値も 5A×4=20A → 1/4 程度に削減できる。低速・大トルク用途では合理的な選択肢。

「トルクは出るが、最高速度が周波数に比例して低くなる」のがインバータ V/f制御の本質的な制約。
30Hz 駆動なら最高速度は定格の半分の 900rpm しか出ない。
巻取り機で高速ラインに対応したい場合は、この速度上限が問題になる。
→ 用途に必要な最高速度を先に確認し、そこから制御方式を選ぶことが設計の鉄則。

この差の意味

同じ「110V」でも、周波数も合わせて下げる（V/f制御）と、磁束密度が定格通り保たれるためトルクは維持されます。
一方、電圧だけ下げる（サイリスタ式）と、磁束密度が落ちてトルクが急減します。
低速で大きなトルクが必要な用途では、インバータV/f制御の優位は決定的です。

なぜサイリスタ式とインバータでこんなに差が出るのか

物理的な理由

磁束密度の維持 モーターのトルクは「磁束密度 × 電流」で決まる。
サイリスタ式：電圧だけ下げる → 周波数は60Hz固定 → 磁束密度が低下 → トルク激減
インバータ V/f制御：電圧と周波数を比例で下げる → 磁束密度は定格を維持 → トルクも維持
電流の流れ方 サイリスタ式：低電圧で同じ周波数（60Hz）のまま使うと、巻線インピーダンスが変わらず大電流が流れる
インバータ V/f制御：周波数も下がるとインピーダンスも下がり、適切な電流に収まる
発熱の挙動 サイリスタ式で大電流が流れる状態 → 巻線発熱大
インバータ V/f制御で適切な電流 → 発熱も大幅減。連続運転に向く
